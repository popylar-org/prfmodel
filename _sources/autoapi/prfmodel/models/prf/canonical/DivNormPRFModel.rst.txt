prfmodel.models.prf.canonical.DivNormPRFModel
=============================================

.. py:class:: prfmodel.models.prf.canonical.DivNormPRFModel(prf_model: prfmodel.models.base.BasePopulationResponse, shared_params: list[str] | None = None, encoding_model: prfmodel.models.base.BaseStimulusEncoder | type[prfmodel.models.base.BaseStimulusEncoder] = PRFStimulusEncoder, impulse_model: prfmodel.impulse.base.BaseImpulse | type[prfmodel.impulse.base.BaseImpulse] | None = DerivativeTwoGammaImpulse, scaling_model: prfmodel.scaling.base.BaseScaling | type[prfmodel.scaling.base.BaseScaling] | None = Baseline, regressors_model: prfmodel.regressors.base.BaseRegressors | list[prfmodel.regressors.base.BaseRegressors] | None = None)



   Divisive normalization population receptive field (pRF) model.

   This class performs divisive normalization between an activation (numerator) and a normalization (denominator)
   pRF response and combines them with an impulse and scaling model. Both responses come from the same model class,
   but their parameters can differ.

   :param prf_model: A population receptive field response model instance.
   :type prf_model: BasePopulationResponse
   :param shared_params: Names of pRF parameters that are shared between the two responses. All names must appear in
                         ``prf_model.parameter_names``.
   :type shared_params: list of str, optional
   :param encoding_model: An stimulus encoding model class or instance. Model classes will be instantiated during initialization.
                          The default creates a :class:`~prfmodel.models.prf.PRFStimulusEncoder` instance.
   :type encoding_model: BaseStimulusEncoder or type, default=PRFStimulusEncoder
   :param impulse_model: An impulse model class or instance. Model classes will be instantiated during
                         initialization. The default creates a :class:`~prfmodel.impulse.DerivativeTwoGammaImpulse`
                         instance with default values.
   :type impulse_model: BaseImpulse or type or None, default=DerivativeTwoGammaImpulse, optional
   :param scaling_model: A scaling model class or instance. Model classes will be instantiated during initialization.
                         The default creates a :class:`~prfmodel.scaling.Baseline` instance.
   :type scaling_model: BaseScaling or type or None, default=Baseline, optional
   :param regressors_model: A regressor model instance, a list of regressor model instances, or `None`. When a list is
                            provided, it is wrapped in a :class:`~prfmodel.regressors.RegressorsList` and its contributions
                            are summed. The regressor contribution is added after the scaling model.
   :type regressors_model: BaseRegressors or list of BaseRegressors or RegressorsList or None, default=None, optional

   .. rubric:: Notes

   The divisive normalization model follows these steps

   1. The two pRF response models make predictions for the stimulus grid.
   2. The encoding model encodes the responses with the stimulus design.
   3. The two encoded responses are combined through divisive normalization.
   4. The combined response is convolved with an impulse response (optional).
   5. The scaling model modifies the convolved response (optional).
   6. The regressors model adds a linear combination of fixed regressors to the scaled response (optional).


   .. py:method:: __call__(stimulus: prfmodel.stimuli.PRFStimulus, parameters: pandas.DataFrame, regressors: pandas.DataFrame | None = None, dtype: str | None = None) -> prfmodel.typing.Tensor

      Predict the combined model response to a stimulus.

      :param stimulus: Population receptive field stimulus object.
      :type stimulus: PRFStimulus
      :param parameters: Dataframe with columns containing different model parameters and rows containing parameter values
                         for different units.
      :type parameters: pandas.DataFrame
      :param regressors: Regressor design data. Required when the canonical model has a regressors model configured.
                         A single data frame with shape `(num_frames, num_regressors)` whose columns cover the names
                         required by every configured regressor model. Extra columns are ignored.
      :type regressors: pandas.DataFrame, optional
      :param dtype: The dtype of the prediction result. If `None` (the default), uses the dtype from
                    :func:`prfmodel.utils.get_dtype`.
      :type dtype: str, optional

      :returns: The predicted model response with shape `(num_units, num_frames)` and dtype `dtype`.
      :rtype: Tensor



   .. py:property:: parameter_names
      :type: list[str]


      Names of parameters used by the model.

      Shared response model parameters appear once (no suffix). Non-shared parameters are suffixed with the two
      response suffixes (e.g. ``_center``/``_surround``). The combination, encoding, impulse and scaling
      parameters are appended.
