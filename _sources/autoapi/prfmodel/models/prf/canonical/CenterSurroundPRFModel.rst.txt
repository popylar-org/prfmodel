prfmodel.models.prf.canonical.CenterSurroundPRFModel
====================================================

.. py:class:: prfmodel.models.prf.canonical.CenterSurroundPRFModel(prf_model: prfmodel.models.base.BasePopulationResponse, encoding_model: prfmodel.models.base.BaseStimulusEncoder | type[prfmodel.models.base.BaseStimulusEncoder] = PRFStimulusEncoder, impulse_model: prfmodel.impulse.base.BaseImpulse | type[prfmodel.impulse.base.BaseImpulse] | None = DerivativeTwoGammaImpulse, scaling_model: prfmodel.scaling.base.BaseScaling | type[prfmodel.scaling.base.BaseScaling] | None = DoGAmplitude, regressors_model: prfmodel.regressors.base.BaseRegressors | list[prfmodel.regressors.base.BaseRegressors] | None = None, change_params: list[str] | None = None)



   Center-surround composite population receptive field model.

   This is a generic class that runs two PRF responses through stimulus encoding and impulse response convolution
   independently, then combines them via a temporal model:
   y(t) = p1(t) * amplitude_center + p2(t) * amplitude_surround + baseline

   The two responses differ in the values of ``change_params``. Each parameter
   ``p`` in that list is split into ``{p}_center`` and ``{p}_surround``.

   :param prf_model: A population receptive field response model instance.
   :type prf_model: BasePopulationResponse
   :param encoding_model: An stimulus encoding model class or instance. Model classes will be instantiated during initialization.
                          The default creates a :class:`~prfmodel.models.prf.PRFStimulusEncoder` instance.
   :type encoding_model: BaseStimulusEncoder or type, default=PRFStimulusEncoder
   :param impulse_model: An impulse model class or instance. Model classes will be instantiated during
                         initialization. The default creates a :class:`~prfmodel.impulse.DerivativeTwoGammaImpulse`
                         instance with default values.
   :type impulse_model: BaseImpulse or type or None, default=DerivativeTwoGammaImpulse, optional
   :param scaling_model: A scaling model class or instance. Model classes will be instantiated during initialization.
                         The default creates a :class:`~prfmodel.scaling.BaselineAmplitude` instance.
   :type scaling_model: BaseScaling or type or None, default=BaselineAmplitude, optional
   :param regressors_model: A regressor model instance, a list of regressor model instances, or `None`. When a list is
                            provided, it is wrapped in a :class:`~prfmodel.regressors.RegressorsList` and its contributions
                            are summed. The regressor contribution is added after the scaling model.
   :type regressors_model: BaseRegressors or list of BaseRegressors or RegressorsList or None, default=None, optional
   :param change_params: Names of the parameters that differ between the center and surround responses.
                         All entries must be present in ``prf_model.parameter_names``.
   :type change_params: list[str], default=["sigma"]

   .. rubric:: Notes

   The center-surround composite model follows these steps:

   1. Two PRF response predictions are computed, one using ``{p}_center`` and one
      using ``{p}_sorround`` for each ``p`` in ``change_params``.
   2. The encoding model encodes each response with the stimulus design.
   3. Each encoded response is optionally convolved with an impulse response.
   4. The two responses are stacked and combined by the scaling model.
   5. The regressors model (optional) adds a linear combination of fixed regressors to the combined response.


   .. py:method:: __call__(stimulus: prfmodel.stimuli.PRFStimulus, parameters: pandas.DataFrame, regressors: pandas.DataFrame | None = None, dtype: str | None = None) -> prfmodel.typing.Tensor

      Predict the composite model response (considering Center and Surround responses).

      Applies the temporal model to the stacked responses from predict_responses.
      When scaling_model=None, returns a simple subtraction (response_center - response_surround)

      :param stimulus: Population receptive field stimulus object.
      :type stimulus: PRFStimulus
      :param parameters: Dataframe with columns containing different model parameters and rows containing parameter values
                         for different units.
      :type parameters: pandas.DataFrame
      :param regressors: Regressor design data. Required when the canonical model has a regressors model configured.
                         A single data frame with shape `(num_frames, num_regressors)` whose columns cover the names
                         required by every configured regressor model. Extra columns are ignored.
      :type regressors: pandas.DataFrame, optional
      :param dtype: The dtype of the prediction result. If `None` (the default), uses the dtype from
                    :func:`prfmodel.utils.get_dtype`.
      :type dtype: str, optional

      :returns: The predicted model response with shape `(num_units, num_frames)` and dtype `dtype`.
      :rtype: Tensor
