prfmodel.models.CFStimulusEncoder
=================================

.. py:class:: prfmodel.models.CFStimulusEncoder



   Encoding model for connective field stimuli.

   Multiplies a source response with a connective model response and sums over the vertices in the source response.



   .. py:method:: __call__(stimulus: prfmodel.stimuli.CFStimulus, response: prfmodel.typing.Tensor, parameters: pandas.DataFrame, dtype: str | None = None) -> prfmodel.typing.Tensor

      Encode a Connective field model response with a source response.

      :param stimulus: Connective field stimulus object.
      :type stimulus: CFStimulus
      :param response: Connective field response.
      :type response: Tensor
      :param parameters: Dataframe with columns containing different model parameters and rows containing parameter values
                         for different units.
      :type parameters: pandas.DataFrame
      :param dtype: The dtype of the prediction result. If `None` (the default), uses the dtype from
                    :func:`prfmodel.utils.get_dtype`.
      :type dtype: str, optional

      :returns: The predicted model response with shape `(num_units, num_frames)` and dtype `dtype`.
      :rtype: Tensor



   .. py:property:: parameter_names
      :type: list


      Does not have any parameters. Returns an empty list.
