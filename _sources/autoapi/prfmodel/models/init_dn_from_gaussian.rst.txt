prfmodel.models.init_dn_from_gaussian
=====================================

.. py:function:: prfmodel.models.init_dn_from_gaussian(gaussian_params: pandas.DataFrame, sigma_ratio: float = 2.0, sigma_normalization: float | None = None, baseline_activation: float | None = None, amplitude_normalization: float = 1.0, baseline_normalization: float = 1.0) -> pandas.DataFrame

   Initialize DN model parameters from fitted Gaussian model parameters.

   Converts the output of a fitted :class:`~prfmodel.models.gaussian.Gaussian2DPRFModel`
   into starting parameters for a :class:`DivNormGaussian2DPRFModel`, suitable for
   subsequent SGD.

   :param gaussian_params: DataFrame of fitted parameters from a ``Gaussian2DPRFModel``.
                           Must contain columns: ``sigma`` and ``amplitude`` (plus all shared columns).
   :type gaussian_params: pandas.DataFrame
   :param sigma_ratio: Ratio used to set the normalization size (phi = sigma_normalization / sigma_activation in the paper):
                       ``sigma_normalization = sigma_activation * sigma_ratio``.
                       Must be >= 1.0, as the normalization pRF must be at least as large as the activation
                       pRF. Ignored when ``sigma_normalization`` is provided.
   :type sigma_ratio: float, default=2.0
   :param sigma_normalization: Fixed normalization size applied to all rows. Must be >= ``sigma`` for every row in
                               ``gaussian_params``. When provided, overrides ``sigma_ratio``.
   :type sigma_normalization: float, optional
   :param baseline_activation: Initial value for ``baseline_activation`` (b in the DN formula). If ``None`` (the
                               default), the value is taken from the ``baseline`` column of ``gaussian_params``; a
                               ``ValueError`` is raised if that column is absent. When an explicit float is given it
                               is always used, regardless of whether a ``baseline`` column is present.
   :type baseline_activation: float, optional
   :param amplitude_normalization: Initial value for ``amplitude_normalization`` (c in the DN formula). Controls the
                                   scaling of the normalization Gaussian.
   :type amplitude_normalization: float, default=1.0
   :param baseline_normalization: Initial value for ``baseline_normalization`` (d in the DN formula). Controls the
                                  normalization baseline in the denominator. Must be > 0 to avoid division by zero.
   :type baseline_normalization: float, default=1.0

   :returns: DataFrame of DN initial parameters with columns:
             ``sigma_activation`` (= ``sigma``), ``sigma_normalization``,
             ``amplitude_activation`` (= ``amplitude``), ``baseline_activation`` (b, = ``baseline``
             if present), ``amplitude_normalization`` (c), ``baseline_normalization`` (d),
             plus all shared columns unchanged. The ``sigma``, ``amplitude``, and ``baseline``
             (if present) columns are dropped.
   :rtype: pandas.DataFrame

   :raises ValueError: If ``sigma_ratio`` is less than 1.0.
   :raises ValueError: If ``sigma_normalization`` is smaller than ``sigma`` for any row in ``gaussian_params``.
   :raises ValueError: If ``baseline_activation`` is ``None`` and ``gaussian_params`` does not contain a
       ``baseline`` column.
