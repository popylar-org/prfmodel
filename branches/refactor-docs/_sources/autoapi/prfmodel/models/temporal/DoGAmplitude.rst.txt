prfmodel.models.temporal.DoGAmplitude
=====================================

.. py:class:: prfmodel.models.temporal.DoGAmplitude



   Linear amplitude model for difference of Gaussians.

   Combines two temporal responses with independent amplitudes and a baseline:
   y(t) = inputs[:, 0] * amplitude_center + inputs[:, 1] * amplitude_surround + baseline

   .. rubric:: Examples

   >>> import numpy as np
   >>> import pandas as pd
   >>> params = pd.DataFrame({
   ...     "amplitude_center": [1.0, -1.0],
   ...     "amplitude_surround": [-0.5, 0.5],
   ...     "baseline": [0.0, 0.1],
   ... })
   >>> num_frames = 10
   >>> inputs = np.ones((params.shape[0], 2, num_frames))
   >>> model = DoGAmplitude()
   >>> resp = model(inputs, params)
   >>> print(resp.shape)
   (2, 10)


   .. py:method:: __call__(inputs: prfmodel.typing.Tensor, parameters: pandas.DataFrame, dtype: str | None = None) -> prfmodel.typing.Tensor

      Predict the model response.

      :param inputs: Input tensor with two temporal responses stacked along axis 1,
                     shape (num_units, 2, num_frames).
      :type inputs: :data:`prfmodel.typing.Tensor`
      :param parameters: Dataframe with columns containing different model parameters and rows containing parameter values
                         for different units.
      :type parameters: pandas.DataFrame
      :param dtype: The dtype of the prediction result. If `None` (the default), uses the dtype from
                    :func:`prfmodel.utils.get_dtype`.
      :type dtype: str, optional

      :returns: The predicted model response with shape `(num_units, num_frames)` and dtype `dtype`.
      :rtype: Tensor



   .. py:property:: parameter_names
      :type: list[str]


      Names of parameters used by the model.

      Parameter names are: ``amplitude_center``, ``amplitude_surround``, and ``baseline``.
